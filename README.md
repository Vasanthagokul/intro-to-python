## üöÄ Getting Started

### Prerequisites
- Python 3.8+ installed
- Jupyter Notebook or Jupyter Lab
- Basic understanding of what programming is

### Setup Instructions

1. **Clone this repository:**
   ```bash
   git clone https://github.com/vasanthagokul/python-intro-labs
   cd python-intro-labs/intro-to-python
   ```

2. **Create a virtual environment (recommended):**
   ```bash
   python -m venv venv
   
   # On Windows:
   venv\Scripts\activate
   
   # On Mac/Linux:
   source venv/bin/activate
   ```

3. **Install Jupyter:**
   ```bash
   pip install jupyter
   ```

4. **Start Jupyter Notebook:**
   ```bash
   jupyter notebook
   ```

5. **Open the notebooks** in order (01, 02, 03...) and follow along!

## üìù How to Use This Repository

- **Work sequentially** - Start with notebook 01 and progress through
- **Run all code cells** - Type `Shift + Enter` to execute cells
- **Experiment freely** - Modify code, try variations, break things!
- **Complete exercises** - Each notebook has practice problems

## üí° Tips for Success

1. **Read explanations carefully** before running code
2. **Type code yourself** instead of copy-pasting (builds muscle memory)
3. **Test your understanding** by modifying examples
4. **Debug actively** - errors are learning opportunities
5. **Take breaks** - Stand up, stretch every 45-60 minutes
